Fáze školení

Klikni na fázi — okamžitě se přepne, co vidí účastníci i projektor. Intro = před školením, Live = otevřené Q&A, Outro = závěr.

🎬 Intro — uvodní obrazovka

Google Slides: File → Share → Publish to web → Embed → zkopíruj src z iframe.
PDF z Google Drive: sdílej veřejně → použij https://drive.google.com/file/d/<ID>/preview.
Canva: Share → More → Embed → zkopíruj URL.
PowerPoint (.pptx): použij tlačítko níže — nahraje se a vygeneruje se URL automaticky.

Agenda dne

Časový rozvrh školení. Zobrazí se v intro fázi i na projektoru během Q&A. Aktuální blok se sám vyznačí podle reálného času.

⏱️ Pauza / odpočet

Spusť veřejný countdown — účastníci i projektor uvidí velkou časomíru.

Materiály ke stažení

Slidy, fotky, PDF s manuálem… Soubory uvidí účastníci v intro a outro fázi a mohou si je stáhnout.

📊 Live polls (hlasování)

Účastníci hlasují tlačítky, projektor zobrazí výsledky live. Jeden aktivní poll v jeden moment — předchozí se sám uzavře, když aktivuješ jiný.

🎵 Hudba na pozadí (projektor)

mp3 / mp4 podtext, který se sám pustí na projektoru podle fáze: intro, pauza (break_until) a outro. Hraje výhradně projektor (?display=1) — účastnické mobily zůstávají potichu. Admin na projektoru musí jednou kliknout „🔊 Spustit zvuk" kvůli autoplay policy.

Účastníci školení

Když je zapnuto, účastník musí před vstupem do Q&A vyplnit formulář (zákaznické číslo Intercars, e-mail, IČ, telefon, firma).
